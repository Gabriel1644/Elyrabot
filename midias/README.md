# 📁 Pasta midias/

Coloque aqui os arquivos de mídia do menu:

| Arquivo       | Tipo     | Descrição                              |
|---------------|----------|----------------------------------------|
| `menu.mp4`    | Vídeo    | Vídeo que aparece no !menu (prioridade)|
| `menu.gif`    | GIF      | GIF animado para o menu                |
| `menu.jpg`    | Imagem   | Foto estática do menu                  |
| `menu.jpeg`   | Imagem   | Alternativa JPG                        |
| `menu.png`    | Imagem   | Alternativa PNG                        |
| `menu.mp3`    | Áudio    | Áudio tocado antes do menu             |
| `menu.ogg`    | Áudio    | Áudio em formato OGG/Opus              |

## Personalização por grupo

Use os comandos no WhatsApp:
- `!setmenumidia` — responda uma imagem/vídeo para definir a mídia do menu deste grupo
- `!setmenunome <nome>` — define o nome do bot neste grupo
- `!setmenuaudio on/off` — ativa/desativa o áudio antes do menu
- `!resetmenugrupo` — reseta todas as personalizações do grupo
